## **Overview**  
This repository contains the LaTeX source code for my resume, inspired by the widely recognized Overleaf **Jake Ryan template**. The resume is designed to be:  
- **Elegant** and visually professional.  
- **Customizable**, with well-structured LaTeX code for easy modifications.  
- **ATS-friendly**, ensuring compatibility with applicant tracking systems.
